Stocked provides a clean and simple user interface for viewing market updates.



Written in JavaScript and HTML using Bootstrap, D3.js, and the Alpha Vantage API.

Stocked

Alpha Vantage

Stocked uses Alpha Vantage to obtain realtime and historical stock data.

D3.js

Stocked uses D3.js to create graphs from raw data.
